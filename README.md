# CleanTrack
## About

The Smart Garbage Collection System is a web application that helps improve the garbage collection process by connecting residents, garbage truck drivers, and administrators on a single platform.

Residents can view garbage collection schedules, receive notifications before the truck arrives, track the truck's live location, and report missed collections or submit complaints.

Drivers can view their assigned routes, update collection statuses, and share their live location.

Administrators can manage users, drivers, trucks, routes, and collection schedules through a web dashboard.

## Features

- User authentication
- View garbage collection schedules
- Live garbage truck tracking
- Arrival notifications
- Report missed collections
- Submit complaints
- Driver route management
- Collection status updates
- Admin dashboard
- User, truck, and route management
